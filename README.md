tails
=====

tail multiple files with colors

### simple install

    curl https://raw.github.com/illb/tails/master/tails > ~/bin/tails; chmod 755 ~/bin/tails

### simple usage
* usage : program filelist

### example
#### command

    tails \
    /var/log/system.log \ 
    /path/to/tomcat/logs/catalina.out
